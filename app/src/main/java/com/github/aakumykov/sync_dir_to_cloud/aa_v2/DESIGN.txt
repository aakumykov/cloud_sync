1) Тчение источника:
CloudReader (специфический для разных хранилищ) - читает из специфического источника,
    выдаёт стандартизированные объекты.
StorageToDatabaseReader - сохраняет эти объекты в БД, добавляя новые или обновляя существующие.

Как создаётся и используется CloudReader

CloudReader <-- CloudReaderFactory <-- (сборка в словарь Dagger-ом) <-- CloudReaderCreator()
